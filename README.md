### Install
```
make install
make build
make package-install
```

#### Usage
Для запуска игр используйте команды

Проверка на четность
```
brain-even
```

Калькулятор
```
brain-calc
```

НОД(наибольший общий делитель)
```
brain-gcd
```

"Простое ли число?"
```
brain-prime
```

Арифметическая прогрессия
```
brain-progression
```
